3. <meta>
الوصف: تُستخدم لتوفير بيانات وصفية (metadata) عن صفحة الويب. البيانات الوصفية لا تظهر للمستخدم مباشرةً، ولكنها تساعد المتصفح ومحركات البحث في فهم الصفحة.

///////////////////////////////////////////

<meta charset="UTF-8">:
تحدد ترميز الأحرف المستخدم في الصفحة (UTF-8 في هذا المثال).
UTF-8 هو أكثر الترميزات شيوعًا لأنه يدعم مجموعة كبيرة من الأحرف، بما في ذلك الحروف العربية.
يضمن ظهور النصوص (مثل اللغة العربية) بشكل صحيح.


//////////////////////////////////////////////////
/////////////////////////////////////////////////
/////////////////////////////////////////////////

<meta name="viewport" content="width=device-width, initial-scale=1.0">:

تُستخدم لجعل الصفحة مناسبة للأجهزة المختلفة (Responsive Design).
السمة viewport تُخبر المتصفح بكيفية التحكم في أبعاد الصفحة وتوسيعها.
تجعل الصفحة تبدو جيدة على الشاشات الصغيرة مثل الهواتف.

How to View HTML Source
------------------------------------------------------------------------------------
---------------------------------------------------------------------------------

View HTML Source Code:
Click CTRL + U in an HTML page, or right-click on the page and select "View Page Source". This will open a new tab containing the HTML source code of the page.

Inspect an HTML Element:
Right-click on an element (or a blank area), and choose "Inspect" to see what elements are made up of (you will see both the HTML and the CSS). You can also edit the HTML or CSS on-the-fly in the Elements or Styles panel that opens.

HTML is Not Case Sensitive
HTML tags are not case sensitive: <P> means the same as <p>.

The HTML standard does not require lowercase tags, but W3C recommends lowercase in HTML, and demands lowercase for stricter document types like XHTML.

At W3Schools we always use lowercase tag names.

HTML ليس حساسًا لحالة الأحرف
علامات HTML ليست حساسة لحالة الأحرف: <P> تعني نفس <p>.

لا يتطلب معيار HTML علامات صغيرة، لكن W3C يوصي باستخدام أحرف صغيرة في HTML، ويتطلب أحرفًا صغيرة لأنواع المستندات الأكثر صرامة مثل XHTML.

في W3Schools نستخدم دائمًا أسماء العلامات الصغيرة.


-----------------------------------------
لماذا يوصى باستخدام الأحرف الصغيرة في HTML؟

XHTML: وهو نوع أكثر صرامة من HTML، يطالب بشكل صارم باستخدام الأحرف الصغيرة.
